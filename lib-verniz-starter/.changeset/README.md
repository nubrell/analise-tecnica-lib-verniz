# Changesets

Este diretório contém os changesets que descrevem mudanças nos pacotes.

## Como usar

1. Execute `yarn changeset` para criar um novo changeset
2. Siga as instruções interativas
3. Commit o arquivo changeset criado
4. O CI/CD detectará e criará um PR de release automaticamente

Para mais informações, consulte a documentação do Changesets.
